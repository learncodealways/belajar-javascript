Public Class Field

  • Public artinya bisa diakses dari luar class
  • Untuk membuat public class filed, kita bisa langsung buat nama field dengan value selevel dengan method
  • Jiika kita tidak memasukkan value ke dalam field tersebut, artinya filed tersebut memiliki value undefined

Kode : Public Class Field

        class Person{
            firstName = 'Muhammad'
            lastName = 'Ramadhan'
            balance = 100

            constructor (firstName, lastName, balance){
                this.firstName = firstName
                this.lastName = lastName
                this.balance = balance
            }
        }

        const rama = new Person()
        console.log(rama)
