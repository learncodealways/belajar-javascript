Super Constructor

  • Class Inheritance sifatnya Prototype Inheritance
  • Bagaimana dengan Constructor Inheritance? Sebenarnya Constructor Inheritance hanyalah melakukan eksekusi constructor lain dengan tujuan agar property di construcotr lain bisa ditambahkan ke instance object ini
  • Dalam kasus ini, jika kita ingin mencapai hasil yang sama, kita bisa menggunakan kata kunci super di dalam construcotr
  • Kata kunci super digunakan untuk memanggil construcotr super class
  • Jika di child class kita membuar construcotr, maka kita wajib memanggil parent walaupun di parent tidak ada construcotr
