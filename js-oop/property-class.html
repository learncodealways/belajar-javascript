Property di class

  • Sama seperti constructor function, dalam class pun kita bisa menambahkan property
  • Karena hasil akhirnya adalah sebuah object, jadi menambahkan property di class bisa juga dilakukan di instance objectnya

Kode : Property di class

        class Person{
            constructor(name){
                this.name = name
            }
        }

        const rama = new Person('rama')
        console.info(rama.name)
