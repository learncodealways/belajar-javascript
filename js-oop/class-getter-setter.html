Getter dan Setter di Class

  • Class juga mendukung pembuatan getter dan setter
  • Perlu diingat, getter dan setter ini akan berada di prototype, bukan di instace object

Kode : Getter dan Setter

        class Person{
            constructor (firstName, lastName, userName){
                this.firstName = firstName
                this.lastName = lastName
                this.userName = userName
            }

            get fullName(){
                return `${this.firstName} ${this.lastName} dan nama panggilan saya adalah ${this.userName}`
            }

            set fullName(value){
                const result = value.split(" ")
                this.firstName = result[0]
                this.lastName = result[1]
                this.userName = result[2]
            }
        }

        const person = new Person('Muhammad', 'Ramadhan', 'Rama')
        console.log(person.fullName)
