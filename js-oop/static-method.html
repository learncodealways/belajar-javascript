Static Method

  • Kata kunci static juga tidak hanya bisa ditambahkan di dield, tapi juga di method
  • Jika kita tambahkan di method, artinya method tersebut jadi milik clasnnyam bukan prototype
  • Dan untuk mengakses method tersebut, kita juga bisa lakukan seperti static class field

Kode : Static Method

        class Penjumlahan{
            static sum(...numbers){
                let total = 0

                for (const number of numbers){
                    total += number
                }
                return total
            }
        }

        const result = Penjumlahan.sum(2,2,2,2,2)
        console.log(result)
