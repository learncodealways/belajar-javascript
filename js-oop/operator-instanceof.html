Operator instanceof

  • Kadang ada kasus kita ingin mengecek apakah sebuah object merupakan instance dari class
  • Kita tidak bisa mengecek operator typeof, karean object dari class, jika kita gunakan operator typeof, hasilnya adalah "object"
  • Operator instanceof akan menghasilkan boolean, true jika benar object tersebut adalah instance objectnya, atau false jika bukan

Kode : Operator instanceof

        class Employee {

        }

        class Manager{

        }

        const rama = new Manager()
        const eko = new Employee()
       
        console.log(rama instanceof Manager) //true (karena rama dibuat dari class Manager)

        console.log(rama instanceof Employee) //false (karena rama tidak dibuat dari class Employee)

        console.log(eko instanceof Employee) //true (karena eko dibuat dari class Employee)

        console.log(eko instanceof Manager) //false (karena eko tidak dibuat dari class Manager)

Operator instanceof di Class Inheritance

  • Operator instanceof mendukung class Inheritance, artinya instanceof juga bisa digunakan untuk mengecek, apakah sebuah object adalah instance dari class tertentu, atau turunan dari class tertentu?

Kode : Operator instanceof Inheritance

           
         class Person {

         }
 
         class General extends Person{
 
         }
 
         const Rama = new General()
         console.log(Rama instanceof Person) // true (karena Rama turunan langsung dari Person atau General extends dengan Person)
         
         console.log(Rama instanceof General) // true (karena Rama dibuat dengan General)
