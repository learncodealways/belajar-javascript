Membuat Object

  • Sebenarnya kita sudah belajar tipe data object, dengan cara membuat variable dengan tipe data object
  • Namun pembuatan object menggunakan tipe data object, akan membuat object selalu unik, sedangkan dalam OOP, biasanya kita akan membuat class sebagai cetakan, sehingga bisa membuat object dengan karaktersitik yang sama berkali-kali, tanpa harus mendeklarasikan object berkali-kali seperti menggunakan tipe data objet

Kode : Membuat Object dengan Object

        const rama = {
            firstName : "Muhammad",
            lastName : "Ramadhan"
        }

        const budi = {
            firstName : "Budi",
            lastName : "Nugraha"
        }

Membuat Constructor Function

  • Sebelum EcmaScript versi 6, pembuatan class, biasanya menggunakan function. Ha ini dikarenakan sebenarnya JavaScript bukanlah bahasa pemrograman yang fokus ke OOP
  • Untuk membuat class di JavaScript lama, kita bisa membuat function
  • Function ini disebut dengan Constructor Function

Kode : Constructor Function

        function Person(){

        }

Membuat Object dari Constructor Function

  • Setelah kita membuat class, jika kita ingin membuat object dari class tersebut, kita bisa menggunakan kata kunci new, lalu diikuti dengan nama constructor functionnya.

Kode : Membuat Object

        function Person(){

        }

        const rama = new Person();
        const budi = new Person();
